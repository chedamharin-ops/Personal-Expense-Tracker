# Personal Expense Tracker

This is a Web Programming Mini-Project that tracks personal expenses, built with HTML, CSS, JavaScript, PHP, and a MySQL Database.

## Features
- Secure Login & Registration
- Add, View, and Delete Expenses
- Total Expense Summaries & Charts
- Data is securely stored in a MySQL backend

## Setup Instructions for Teachers / Evaluators

1. **Install XAMPP** (if not already installed).
2. Move this entire `mini_project` folder into the `C:\xampp\htdocs\` directory.
3. Open the **XAMPP Control Panel** and start **Apache** and **MySQL**.
4. Open your web browser and go to `http://localhost/phpmyadmin/`.
5. Click on **Databases** and create a new database named `expense_tracker`.
6. Click on your newly created `expense_tracker` database on the left sidebar.
7. Go to the **Import** tab at the top.
8. Click **Choose File** and select the `expense_tracker.sql` file located in this project folder.
9. Scroll down and click **Go** to import the tables.
10. Finally, open a new browser tab and go to: `http://localhost/mini_project/login.html` to run the project.
